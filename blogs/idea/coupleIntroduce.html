# 缘起

以前时常冒出奇怪的想法，不靠谱也不切合实际，当然那是在实现他之前。或许许多现存的应用就源于许许多多不切实际的想法吧。这些想法虽没机会实现，现在有了博客，可以先把这些奇怪的想法记录下来。

# 概述

解决目前男女相亲的问题，提升匹配度和成功率、扩大接触面

# 背景

目前社会普遍存在结婚生子意愿低，初婚年龄偏大的现象，这可能是人类灭亡的前兆暂且不论。不敢说解决人类繁衍基因传递的大问题，但解决一些现实的男女问题或许有可能，让男女双方在尽可能大而合理的范围内，相对匹配到合适的婚姻对象。相对，因为男女感情真的是一个玄学，有很多在外人看来不可思议的搭配，也有看起来很合适却走不到一起的。上述情况可能更多是自主恋爱，本方案主要考虑相亲并以结婚为目标的情况，相对可能更容易量化。

# 当前问题

当前相亲主要有两种方案，一种就是普遍所知的相亲网站，但目前相亲网站虚假信息横行，非常不靠谱。另一种就是亲朋介绍，民间也有一些红娘掌握了一些相亲对象的信息，这个相对信息比较靠谱，介绍人都是通过一定的关系网，但问题是范围有一定的局限性。如果有一款应用，既能解决资料可靠性问题，又能扩大交往范围，就可以替代上述两种方案，成为新的解决方案。

# 方案描述

简而言之，以APP或网站为载体，也可以是类似公众号，通过社交网站的形式登录和保存每个人的信息。信息尽量量化，从而可以基于推荐系统的人工智能算法，对资料进行匹配。难点在于积累匹配的数据，成功配对的数据越多，就可以更好的计算出什么情况更容易匹配成功。所谓O2O就是组织线下活动，通过广告赞助形成盈利，如果男女双方步入婚姻，可以进一步提供一条龙服务，建立从婚纱照到婚礼等一系列的生态。

# 结论

以上方案纯属胡言乱语一片臆想，完全没有可行性，请权当是痴人说梦。万一某个扎克伯格似的人物，无意看到本文并实现了这个方案，请勿忘向本文作者交纳版权费。最后讲一个关于相亲匹配的心理学小实验结束。男女各100人，后背写有1到100的数字，但是本人不知道自己的数字。实验开始后，在一个小时时间内，要求每个人找一个异性匹配，最后发放配对成功两个人数字相加10倍的奖金。可想而知，数字越大的人，周围围的人越多，他自知背号很大，却总想找到数字更大的人，当然他也不知道自己的背号是多少。而数字小的人，自然无人问津。一个小时的限定是很总要的，数字小的人往往会很快找一个跟自己差不多的人匹配，或者以交易的方式比如请吃晚饭获得愿意匹配的对象。数字在中间的，往往最容易和自己差不多的匹配在一起。往往是数字很高的人，开始有很多人围绕，但随着大多数人都慢慢匹配成功了，他的选择会越来越少。100或99的最后可能会选择一个70或80的进行匹配，而之前围绕在他身边有90多的此时已经有了配对。只是实际当中，每个人都有心中的评判标准，人也不能完全用数字去量化，祝愿每个人都能找到那个合适的人。
