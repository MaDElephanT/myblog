SpringSecurity前后端分离解决方案

Smart 2020-08-13 SpringSecurity前后端分离

# 概述

在平台开发中，安全认证、权限管理是一个很重要的功能。需要对用户登录进行验证，根据用户信息返回用户权限，并管理和跟踪用户登录状态。一般基于SpringBoot或SpringCloud框架的安全管理，会引入Spring家族的Spring Security实现权限管理的功能。但是如果是基于SpringBoot和Vue的前后端分离项目，Spring Security便不能直接拿来使用。需要对Spring Security进行一定的改造。

# 涉及技术

SpringBoot SpringSecurity Vue

# 方案设计

前端基于Vue，后端基于SpringBoot整合SpringSecurity

# 前端设计

用户登录界面，用户填写用户名密码提交到后台；动态菜单生成，根据后台返回的菜单列表生成对应登录用户权限的页面侧边栏菜单。

# 后端设计

由SpringSecurity接收用户登录提交的表单并进行验证，根据验证结果返回登录失败或成功，如果登陆成功，还要根据用户信息返回对应的用户权限列表。
